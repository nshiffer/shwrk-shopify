Under the Hood

Built right.

Admin GraphQL API

Uses the Shopify Admin GraphQL API to crawl your store content. Batched link resolution with built-in rate limiting ensures efficient scanning without hitting API limits.

Zero Storefront Impact

All scanning happens server-side. No JavaScript injected into your theme. No impact on page load times or Lighthouse scores. Your customers never notice a thing.

Privacy-First

Only reads your store content to check links. Stores only issue data and scan results. No customer data is accessed or stored. Full GDPR compliance built in.

Permissions

What LinkGuard accesses.

We request only the scopes necessary for the app to function. Nothing extra.

read_products
Scan product descriptions and content for broken internal links. Detect links to archived or deleted products.
read_content
Scan pages and blog posts for broken links. Identify references to draft or unpublished content.
read_online_store_navigation
Check navigation menus for links that point to missing or inactive destinations.
write_url_redirects
Create URL redirects to fix broken links directly from the app. Only used when you explicitly choose to create a redirect.
